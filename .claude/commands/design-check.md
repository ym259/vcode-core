# /design-check - デザインチェック

現在のUIがデザインシステムのルールに従っているか確認します。

## チェック項目

以下の項目を `src/` ディレクトリ内のすべてのコンポーネントで確認してください：

### 1. コンポーネント使用
- [ ] shadcn/ui コンポーネントを使っているか（カスタムのボタンやカード等を自作していないか）
- [ ] Lucide React のアイコンのみ使用しているか

### 2. カラー
- [ ] ハードコードされた色（`text-[#xxx]`, `bg-[#xxx]`）がないか
- [ ] CSS変数（`text-foreground`, `bg-background`, etc.）を使っているか

### 3. スペーシング
- [ ] 8pxグリッドに沿っているか（`p-2`, `p-4`, `p-6`, `p-8`）
- [ ] 奇数のスペーシング（`p-3`, `p-5`, `p-7`）を不必要に使っていないか

### 4. タイポグラフィ
- [ ] フォントウェイトが2種類以内か（`font-normal` + `font-semibold`）
- [ ] 見出しにセマンティックなHTML要素（`h1`-`h4`）を使っているか

### 5. レイアウト
- [ ] レスポンシブ対応されているか（`sm:`, `md:`, `lg:` ブレークポイント）
- [ ] コンテンツ幅に `max-w-5xl` か `max-w-6xl` を使っているか

### 6. シャドウ・ボーダー
- [ ] `shadow-sm` 以下のシャドウのみ使っているか
- [ ] `rounded-lg` を統一的に使っているか

### 7. 日本語テキスト
- [ ] すべてのUIラベルが日本語か
- [ ] 適切な行間（`leading-relaxed` 以上）が設定されているか

## 出力形式

問題がなければ：
「✅ デザインチェック完了 — 問題ありません」

問題がある場合は、ファイル名と行番号、修正提案を一覧で表示してください。
