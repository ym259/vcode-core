# /build — ビルドチェック

`npm run build` を実行し、結果を日本語で報告します。デプロイ前の簡易チェックとして使います。

## 手順

1. `npm run build` を実行する
2. 結果に応じて報告する：

### 成功した場合
- 「ビルド成功 ✅」と伝える
- ビルド時間とページ数を報告する

### 失敗した場合
- エラーメッセージを日本語で解説する
- よくあるエラーパターン：
  - TypeScript 型エラー → 該当ファイルと行番号を示して修正案を提示
  - Module not found → インポートパスの大文字小文字を確認
  - ESLint エラー → 該当ルールと修正方法を説明
- 修正が簡単な場合は修正するか確認する
- 複雑な場合は `/fix` を案内する
