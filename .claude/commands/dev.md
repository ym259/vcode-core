# /dev — 開発サーバー起動

開発サーバーを起動してブラウザで開きます。

## 手順

1. まず `npm run dev` をバックグラウンドで実行する
2. サーバーが起動したら（"Ready" メッセージを確認）、ユーザーに報告する
3. ブラウザで http://localhost:3000 を開く（Chrome MCP が使える場合）

## 注意

- 既にポート3000でサーバーが動いている場合は、その旨を伝えて再起動するか確認する
- 起動に失敗した場合は `/fix` を案内する
- サーバーはバックグラウンドで動き続ける。停止したい場合はユーザーに `Ctrl+C` またはターミナルでの停止方法を案内する
