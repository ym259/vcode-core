# /todo-prod — 本番化タスク一覧

コードベース内の `TODO(prod)` コメントを全スキャンし、本番リリースまでに対応が必要な項目を一覧表示します。

## 手順

1. プロジェクト全体（`src/` 配下）で `TODO(prod)` を含むコメントを検索する
2. 結果を以下の形式でテーブル表示する：

| ファイル | 行 | 内容 |
|---------|-----|------|
| `src/app/api/chat/route.ts` | 12 | Add rate limiting to this endpoint |

3. 見つかった件数のサマリーを出す（例：「本番化に向けて 5 件の TODO があります」）
4. TODO が 0 件の場合は「本番化に向けた TODO はありません 🎉」と伝える
5. 必要に応じて、優先度の高い項目（セキュリティ、データ整合性）をハイライトする
